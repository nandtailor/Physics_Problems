
### **Given Data**
* Length of the string ($L$) = **80 cm**
* Number of antinodes ($n$) = **4**

*(Note: In a standing wave on a string fixed at both ends, the number of antinodes corresponds to the number of loops).*

---

### **The Formula**
For a standing wave on a string fixed at both ends, the total length of the string ($L$) is an integer multiple of half-wavelengths ($\frac{\lambda}{2}$). The formula is:

$$L = n \left( \frac{\lambda}{2} \right)$$

Where:
* $L$ = Length of the string
* $n$ = Number of antinodes (loops)
* $\lambda$ = Wavelength

---

### **Calculation**
Substitute the given values into the formula:

$$80 = 4 \left( \frac{\lambda}{2} \right)$$

Simplify the right side:

$$80 = 2\lambda$$

Isolate $\lambda$ to find the wavelength:

$$\lambda = \frac{80}{2}$$

$$\lambda = 40$$

---

### **Final Answer**
The wavelength of the standing wave is **40 cm**. 

*(Logical check: 4 antinodes mean there are 4 loops. If 4 loops span 80 cm, each loop is 20 cm long. Since one full wavelength consists of 2 loops, the wavelength is $2 \times 20$ = 40 cm).*
